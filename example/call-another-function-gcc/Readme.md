This folder contains an example of calling another function


# Environment: 

OS: *amd64/centos:8*

``` bash
docker run -it amd64/centos:8
```

compiler: gcc

# Compile

``` bash
# generate object file
gcc -o main ./main.c

# generate assembly file
gcc -S ./main.c -masm=intel

# assemble from assembly file
gcc -o main ./main.s
```
